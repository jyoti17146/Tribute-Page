# Tribute Page: Osho

A visual and informational tribute page built to practice web layout fundamentals. This project focuses on presenting a biography and key milestones in a way that is easy to read and aesthetically pleasing.

## Project Aim
The objective of this project was to practice 'Front-end' layout skills. I focused on how to arrange large blocks of text and images effectively, ensuring that the design remains simple and focuses on the subject matter without becoming cluttered.

## Features
*   **Structured Content:** Uses semantic 'HTML' to organize a life timeline and key teachings.
*   **Responsive Layout:** Designed to ensure the text and images scale correctly on mobile and desktop.
*   **Styled Typography:** Focuses on readability and a clean visual hierarchy using 'CSS'.
*   **Asset Management:** Organized use of 'images' and external fonts to enhance the user experience.

##  File Structure
*   `index.html`: The core structure and content of the page.
*   `style.css`: All design rules, including 'Flexbox' and media queries.
*   `images/`: Directory containing all visual assets used in the project.

## Future Enhancements
*   Adding an interactive 'Gallery' section for more visual depth.
*   Implementing a 'Search' feature for specific quotes.
*   Adding subtle 'CSS Animations' for a smoother transition between sections.
